My experience with git

Saturday April 27, 2019

Welcome to my blog. Today I am working on an indepth look at Git and Github. I am excited to be learning this since it is required for my group project. We are going to work on forks from a main project on Github using our local development environments and then combine them back up into the master branch.

I was noticing that I have never build the application database out-of-band with the rest of a web app. This means that I might be learning how to integrate some very tightly coupled components of a web application through the use of Git. It is very interesting indeed. Right now I am working on a fake blog with a little text so that I can have something to practice with during my lessons.
